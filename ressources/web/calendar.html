Documentation du calendrier

URI : /api/calendar

Ce calendrier permet d'extraire des informations sur les animés du calendrier Nautiljon (https://www.nautiljon.com/animes/été-2021.html).

Voici un exemple de résultat 

{
    "animes_amount":1,
    "animes":[
        {
            "title":"One Piece",
            "anime_page":"https://www.nautiljon.com/animes/one+piece.html",
            "type":"Série TV",
            "episodes":"?",
            "kinds":[
                "Comédie",
                "Aventure",
                "Drame",
                "Tournois",
                "Fantasy",
                "Shônen",
                "Fantastique",
                "Action",
                "Guerre",
                "Pirates",
                "Super pouvoirs",
                "Combats",
                "Amitié"
            ],
            "simulcasts":[
                "crunchyroll",
                "adn"
            ],
            "image":"https://www.nautiljon.com/images/anime/00/60/mini/one_piece_6.jpg?11632163624",
            "company":"Toei Animation",
            "description":"Il fut un temps où Gold Roger était le plus grand de tous les pirates, le \"Roi des Pirates\" .... libérer le Pays des Wa du contrôle de Kaido et de son shogun.",
            "rate":"8.87/10",
            "start_date":"1999-10-20",
            "end_date":"?"
        }
    ]
}

animes_amount : Le nombre d'animés dans le résultat

animes : Tableau contenant toutes les informations sur les animés

title : le titre de l'animé

anime_page : L'url permettant d'accéder à la page Nautiljon de l'animé

type : Le type de format (série tv, film, etc...)

episodes : Le nombre d'épisode. Si il n'y a pas de nombre d'épisode, alors la valeur est "?"

kinds : Tableau contenant les genres de l'animé

simulcasts : Tableau contenant les diffuseurs si il y en a des spécifiques

image : L'url permettant d'avoir l'affiche de l'animé

company : L'éditeur de l'animé

description : La description de l'animé

rate : La notation des gens sur l'animé (format 10/10). Si il n'y a pas de note, alors la valeur est "-/10"

start_date : La date du jour de sortie de l'animé (format YYYY-MM-DD)

end_date : La date de fin de diffusion de l'animé (format YYYY-MM-DD). Si il n'y a pas de date, alors la valeur est "?"

Paramètres

Pour filtrer les informations, utilisez les paramètres suivants dans l'url (GET)


title

?title=test : filtrera tout les titres d'animés contenant "test"

Type : String (non sensible à la casse)

type

?type=film : filtrera les animés dont le format est le "film"

Type : String (non sensible à la casse)

    Valeurs possibles
  • all : tout type de format (par défaut)
  • serie_tv : seulement les Séries télévisées
  • oav : seulement les OAV (Original Video Animation)
  • film : seulement le films
  • special : seulement les spéciaux
  • ona : seulement les ONA (Original Net Animation)
  • court-metrage : seulement les courts métrages

limit

?limit=10 : permet de limiter le nombre de résultat. par défaut il est de "30" (et le maximum est de "150")

Type : Integer

sort

?sort=date : permet de trier les résultats

Type : String (non sensible à la casse)

    Valeurs possibles
  • date : trier par date (par défaut)
  • popularity : trier par popularité

public

?public=all : affichera tout type de contenu, les contenus "soft" et les contenus pornographiques

Type : String (non sensible à la casse)

    Valeurs possibles
  • all : tout type de contenu (soft et pornographique)
  • all_without_hentai : tout contenu sans pornographie (par défaut)
  • only_hentai : seulement les contenus pornographiques

eq_episode

?eq_episode=12 : (equal_episode) filtrera que les animés dont le nombre d'épisode est égal à "12" épisodes

Type : Integer

sp_episode

?sp_episode=12 : (superior_episode) filtrera que les animés dont le nombre d'épisode est égal ou supérieur à "12" épisodes

Type : Integer

if_episode

?if_episode=12 : (inferior_episode) filtrera que les animés dont le nombre d'épisode est égal ou inférieur à "12" épisodes

Type : Integer

simulcast

?simulcast=adn : filtrera tout les animés dont l'un des diffuseurs est "ADN"

Type : String (non sensible à la casse)

    Valeurs possibles
  • all : tout types de diffuseurs (par défaut)
  • adn : l'un des diffuseurs est ADN
  • netflix : l'un des diffuseurs est NETFLIX
  • crunchyroll : l'un des diffuseurs est CRUNCHYROLL
  • wakanim : l'un des diffuseurs est WAKANIM
  • youtube : l'un des diffuseurs est YOUTUBE

start_date

?start_date=2021-07-02 : filtrera tout les animés dont le début de diffusion est le "2021-07-02" (YYYY-MM-DD)

Type : Date

eq_rate

?eq_rate=8.5 : (equal_rate) filtrera que les animés dont la note de l'animé est égal à "8.5"/10

Type : Float

sp_rate

?sp_rate=8.5 : (superior_rate) filtrera que les animés dont la note de l'animé est égal ou supérieur à "8.5"/10

Type : Float

if_rate

?if_rate=test : (inferior_rate) filtrera que les animés dont la note de l'animé est égal ou inférieur à "8.5"/10

Type : Float
